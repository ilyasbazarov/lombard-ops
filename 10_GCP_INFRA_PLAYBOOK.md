Драйвер, особенности диалекта, чтение RDB$-метаданных для guard, кодировка подключения. Заполнить по итогам первого рабочего подключения.

Включение, email/password провайдер, custom claims для ролей, проверка токена в API, создание пользователей без self-signup.
